# Симулятор Роевого Интеллекта Дронов

Интерактивная веб-симуляция поведения роя дронов с использованием алгоритмов роевого интеллекта.

## Описание

Этот проект демонстрирует поведение роя дронов, реализуя классические алгоритмы роевого интеллекта:
- Разделение (Separation) - дроны избегают столкновений друг с другом
- Выравнивание (Alignment) - дроны стремятся двигаться в одном направлении с соседями
- Сплочённость (Cohesion) - дроны стремятся оставаться рядом с соседями

Симуляция позволяет в реальном времени изменять параметры поведения роя и наблюдать за результатами.

## Возможности

- Интерактивное управление количеством дронов (от 10 до 200)
- Настройка поведенческих параметров:
  - Максимальная скорость дронов
  - Радиус восприятия окружающих дронов
  - Веса трех основных компонентов поведения (разделение, выравнивание, сплочённость)
  - Притяжение к курсору мыши
- Визуальные настройки:
  - Размер дронов
  - Длина отображаемого следа
- Плавная анимация со счётом FPS
- Адаптивный дизайн, работающий на различных устройствах

## Управление

- Наведите курсор мыши на область симуляции, чтобы притянуть дронов к курсору
- Используйте панель управления (кнопка ⚙️ в правом верхнем углу) для изменения параметров
- Кнопка "Сбросить параметры" возвращает все значения к стандартным

Проект реализован полностью на клиентской стороне с использованием:

## Алгоритм работы

Каждый дрон в симуляции обладает следующими характеристиками:
- Позиция (x, y)
- Скорость (vx, vy)
- Ускорение (ax, ay)
- История перемещения (для отрисовки следа)

На каждом кадре анимации каждый дрон:
1. Определяет соседей в пределах радиуса восприятия
2. Рассчитывает три компонента поведения:
   - Разделение: отталкивание от близко расположенных соседей
   - Выравнивание: согласование направления движения с соседями
   - Сплочённость: движение к центру масс группы соседей
3. При необходимости применяет притяжение к курсору мыши
4. Ограничивает свою скорость максимальным значением
5. Обновляет свою позицию
6. Обрабатывает переход через границы экрана (телепортация на противоположную сторону)
